# Starch Pay Ledger
This library manages and stores StarchPay's Blockchain State


Add to Go project
```
go get github.com/abstractpotato/starch-pay-ledger
```

In Go code
```go
import Ledger "github.com/starch-pay-ledger/disk"
```
