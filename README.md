# Starch Pay Ledger
This library manages storage and retrieval of data structs for StarchPay

Add to Go project
```
go get github.com/abstractpotato/starch-pay-ledger
```

In Go code
```go
import Ledger "github.com/starch-pay-ledger/disk"
```
